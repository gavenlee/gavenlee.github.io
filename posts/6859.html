2.数据库命名规则

每个地方都有自己的规则，例：表名： 模块字母开头_表名 列名： 表名__字段名(便于表多的时候操作数据)

程序设计(待补充)

1.如何处理n对n的关系

1.在声明bean对象的时候持有含对方的对象实现n对n。

2.在数据库里设置虚拟列，用虚拟列存储持有对象的值（不推荐）

2.合多写点方法类，工具类

有些东西重复用的挺多的，能写成工具类就写成工具类，很省事，比如获取当前日期，判断非空，类型转换，单位数值类型处理等。

3.ajax入参分开

Ajax后台请求在controller层里的入参要跟第一次请求的入参分开来写，在执行入参的时候记得把该入参值赋给ajax的入参值。

4.数据库查出来的值最好都用vo类封装

如果使用了mybtis。

每个出参最好都用单独的bean封装， 这样维护的时候也好一些。

    // 计算今日需发货总量，并插入到页面
    calculation (carChartData) {
        let num = 0;
        const div = document.createElement('div');
        for (let i = 0; i < carChartData.length; i++) {
            carChartData[i].value ? num += carChartData[i].value : num += 0;
        }

        div.innerText = '今日需发货数量：' + num;
        div.className = 'total-num';
        document.getElementsByClassName('loading-chart')[0].appendChild(div);

        //处理装车进度条及进度条提示文本
        let loadedNum = 0;
        carChartData.forEach(item => {
            if (item.name === '已装车') {
                loadedNum = item.value;
            }
        })
        let result = Math.floor(loadedNum / num * 100);
        this.$speedOfProgress[0].innerText = `${result}%`;
        if (result >= 98) {
            this.$speedOfProgress.css('left', `${result - 2}%`);
            this.$progressBar.css('width', `${result}%`);
        } else {
            this.$progressBar.css('width', `${result + 2}%`);
            result < 5 ? this.$speedOfProgress.css('left', `${result + 5}%`) : this.$speedOfProgress.css('left', `${result + 1}%`);
        }
    },
    // 当表格请求数据为空时要展示的内容
    renderSpace (dom, del, text) {
        if(del) {
            dom.find('.not-data').remove();
            return;
        }
        text = text || '今天还没有数据呢~~~'
        dom.find('.not-data').remove();
        const div = $(`<div class="not-data">${text}</div>`);
        div.css({
            width: '100%',
            height: '100%',
            display: 'flex',
            justifyContent: 'center',
            alignItems: 'center'
        })
        dom.append(div);
    },
    // 清除最后一次请求到的数据生成的dom
    clearDom(dom, type) {
        dom.find(type).remove();
    },
    bindEvent () {

        window.onresize = () => {
            this.orderChartInit && this.orderChartInit.resize();
            this.carChartInit && this.carChartInit.resize();
            this.monthChartInit && this.monthChartInit.resize();
        }
    },
}
